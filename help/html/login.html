Acceso al Sistema (Login)

Descripción general

Permite acceder a la aplicación seleccionando el rol (Administrador, Técnico, Gestor o Usuario) desde el menú desplegable (combobox).

Errores comunes:
  • Credenciales incorrectas: Se mostrará un aviso de error de autenticación.
  • Rol incorrecto: Si intentas entrar como técnico sin tener permisos, el sistema denegará el acceso.
