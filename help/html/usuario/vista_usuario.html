Panel de Usuario

Desde aquí puedes gestionar tus propias incidencias.

  • Listar por ID: Usa la lupa para buscar una incidencia concreta.
  • Solicitar reapertura: Si una incidencia cerrada persiste, pulsa el botón de enviar para pasarla a estado 'alta'.
